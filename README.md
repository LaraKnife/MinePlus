MinePlus ✨
</div>

MinePlus es un Add-on para Minecraft Bedrock que busca mejorar la calidad de vida en la agricultura y la recolección de materiales, diseñado para ser ligero, eficiente y prevenir el lag.

🚀 Características
Recolección Automática: Los cultivos y bloques se recogen directamente en tu inventario al romperlos (si tienes espacio).

Replantado Inteligente: Los cultivos maduros se replantan automáticamente después de ser cosechados.

Configuración Personalizada: Cada jugador puede activar o desactivar las funciones de asistencia mediante un menú visual in-game sin afectar a los demás.

Multiplayer Friendly: Totalmente compatible con servidores y mundos compartidos.

🎮 Cómo usarlo
Para desactivar o volver a activar las caracteristicas del Add-on, solo necesitas escribir el siguiente comando en el chat del juego:

!menu

Esto abrirá un menú visual donde podrás alternar el interruptor (toggle) según tu preferencia personal.

🛠️ Instalación
Para jugadores:
Descarga el archivo MinePlus.mcpack desde la sección de Releases. (*Proximamente)

Haz doble clic en el archivo o ábrelo con Minecraft.

El juego importará el paquete automáticamente.

Asegúrate de activarlo en los "Comportamientos" (Behavior Packs) del mundo.

Para desarrolladores:
Si deseas compilar el código fuente por tu cuenta:

Clona este repositorio: git clone https://github.com/LaraKnife/MinePlus.git

Instala las dependencias: npm install

Usa npm run deploy para mover el mod a tu carpeta de desarrollo de Minecraft (Windows).

📋 Requisitos
Minecraft Bedrock Edition (versión 1.21.0 o superior).
* No está planeada compatibilidad con versiones anteriores.

Experimental Gameplay: Asegúrate de tener activada la opción "Beta APIs" en la configuración de experimentos del mundo.
