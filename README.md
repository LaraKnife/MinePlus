## 🌎 English Version

MinePlus is a Minecraft Bedrock Add-on designed to improve the Quality of Life (QoL) in farming and resource gathering. It is built to be lightweight, highly efficient, and lag-preventative.

### 🚀 Features

- **Auto-Collection:** Broken blocks and harvested crops go directly into your inventory (if you have enough space), acting as an item vacuum.
- **Smart Replanting:** Fully grown crops are automatically replanted instantly after being harvested.
- **Home System:** Set a personal spawn point anywhere in the world and teleport back to it whenever you need to.
- **Custom Configuration:** Each player can toggle their own assist features using an in-game visual UI, without affecting other players on the server.
- **Multiplayer Friendly:** Fully compatible with Realms, servers, and shared worlds.

### 🎮 How to use

Simply type the following commands in the game chat to access the Add-on's features:

- `!menu` : Opens a visual UI where you can toggle the Auto-Collection and Replant features on or off.
- `!sethome` : Saves your current exact location and dimension as your personal home.
- `!home` : Teleports you instantly to your saved home location.

### 🛠️ Installation

#### For Players:

1. Download the MinePlus.mcpack file from the Releases section.
2. Double-click the file (or open it with Minecraft) to import it automatically.
3. Apply it to your world's Behavior Packs.

#### For Developers:

_If you want to compile the source code yourself:_

1. Clone this repository: `git clone [https://github.com/LaraKnife/MinePlus.git](https://github.com/LaraKnife/MinePlus.git)`
2. Install dependencies: `npm install`
3. Move the mod to your Minecraft development folder (Windows): `npm run deploy`

### 📋 Requirements

- **Minecraft Bedrock Edition (v1.21.0 or higher):** Backwards compatibility is not planned due to recent engine changes.
- **Experimental Gameplay:** You MUST enable Beta APIs in your world's experiment settings for the code to run.

## 🌎 Versión en Español

MinePlus es un Add-on para Minecraft Bedrock que busca mejorar la calidad de vida (QoL) en la agricultura y la recolección de materiales. Está diseñado para ser ligero, eficiente y prevenir el lag.

### 🚀 Características

- **Recolección Automática:** Los cultivos y bloques se recogen directamente en tu inventario al romperlos (si tienes espacio), funcionando como una aspiradora.
- **Replantado Inteligente:** Los cultivos maduros se replantan automáticamente en cuanto son cosechados.
- **Sistema de Home:** Establece un punto de aparición personal en cualquier lugar del mundo y teletranspórtate a él cuando lo necesites.
- **Configuración Personalizada:** Cada jugador puede activar o desactivar sus funciones de asistencia mediante un menú visual in-game sin afectar a los demás.
- **Multiplayer Friendly:** Totalmente compatible con Realms, servidores y mundos compartidos.

### 🎮 Cómo usarlo

Solo necesitas escribir los siguientes comandos en el chat del juego para acceder a las funciones:

- `!menu` : Abre un menú visual donde podrás alternar el interruptor (toggle) de la autorrecolección según tu preferencia.
- `!sethome` : Guarda tu ubicación y dimensión actual como tu hogar personal.
- `!home` : Te teletransporta de manera instantánea a la ubicación de tu hogar guardado.

### 🛠️ Instalación

#### Para Jugadores:

1. Descarga el archivo MinePlus.mcpack desde la sección de Releases.
2. Haz doble clic en el archivo o ábrelo con Minecraft para que el juego lo importe automáticamente.
3. Asegúrate de activarlo en los Paquetes de Comportamientos (Behavior Packs) de tu mundo.

#### Para Desarrolladores:

_Si deseas compilar el código fuente por tu cuenta:_

1. Clona este repositorio: `git clone [https://github.com/LaraKnife/MinePlus.git](https://github.com/LaraKnife/MinePlus.git)`
2. Instala las dependencias: `npm install`
3. Usa el siguiente comando para mover el mod a tu carpeta de desarrollo (Windows): `npm run deploy`

### 📋 Requisitos

- **Minecraft Bedrock Edition (versión 1.21.0 o superior):** No está planeada la compatibilidad con versiones anteriores debido a cambios recientes en el motor.
- **Modo Experimental:** Asegúrate de tener activada la opción Beta APIs (APIs Beta) en la configuración de experimentos del mundo para que el add-on funcione.
